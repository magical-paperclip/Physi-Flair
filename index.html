About sics-ground

sics-ground is an interactive physics sandbox powered by Matter.js where you can play with realistic object physics.

Add circles, squares, and polygons to the playground, watch as they interact using one of four effects: Bounce, Explode, Stick, or Gravity Shift.

This was my first project that I've made using js so I truly hope sics-ground is fun!
